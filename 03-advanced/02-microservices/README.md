# 微服务架构示例

本示例演示如何使用 Caddy 部署和管理微服务架构应用。

## 学习目标

- 理解微服务架构模式
- 掌握 Caddy 在微服务中的应用
- 学会服务注册和发现
- 实现服务间通信和负载均衡

## 架构概览

```
┌─────────────────┐
│   Caddy Gateway │
├─────────────────┤
│ ┌─────┐ ┌─────┐ │
│ │User │ │Order│ │
│ │Service Service│ │
│ └─────┘ └─────┘ │
│ ┌─────┐ ┌─────┐ │
│ │Product Payment│ │
│ │Service Service│ │
│ └─────┘ └─────┘ │
└─────────────────┘
```

## 运行示例

1. 启动所有微服务
2. 启动 Caddy 网关
3. 测试服务间调用

## 学习要点

1. 微服务提高了系统的可扩展性
2. 每个服务独立部署和扩展
3. 网关统一处理跨服务请求
4. 服务发现和健康检查很重要
