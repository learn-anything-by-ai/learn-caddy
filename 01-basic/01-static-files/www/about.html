什么是 Caddy？

Caddy 是一个强大、企业级的开源 Web 服务器，具有自动 HTTPS 功能。它以配置简单、功能强大而著称。

主要优势

🔒 自动 HTTPS

Caddy 自动获取和更新 Let's Encrypt 证书，无需手动配置。

⚡ 高性能

使用 Go 语言编写，性能优异，内存占用低。

🔧 易于配置

Caddyfile 语法简洁直观，配置文件易读易写。

🧩 模块化

丰富的中间件和插件系统，可扩展性强。

应用场景

  • 静态网站托管
  • 反向代理
  • API 网关
  • 负载均衡
  • 微服务架构
