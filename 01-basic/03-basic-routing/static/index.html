📚 路由配置说明

🎯 当前配置的路由规则：

  • / - 主页，显示当前演示页面
  • /api/v1/* - 重写为 v2 API
  • /api/v2/* - 直接响应 API 信息
  • /blog/* - 移除前缀，服务博客文件
  • /admin/* - 需要令牌访问
  • /images/* - 代理到外部图片服务
  • /health - 健康检查端点
  • /version - 版本信息端点
handle /api/v1/* { # 将 v1 API 重写为 v2 rewrite * /api/v2{path} respond "API v1 已重定向到 v2: {path}" 200 } handle /blog/* { # 移除 /blog 前缀 rewrite * {path_regexp ^/blog(/.*)$ $1} root * static/blog file_server } # 重定向示例 redir /old-page /new-page.html 301
